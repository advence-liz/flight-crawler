# 航班管理功能 - 快速参考卡

## 🚀 快速开始（30秒）

```bash
# 1. 启动应用
./start.sh

# 2. 打开浏览器
http://localhost:5173/flight-management

# 3. 开始使用！
```

## 📌 核心功能速查

### 查询航班
1. 选择筛选条件（可选）
2. 点击"查询"按钮
3. 查看结果表格

**支持的筛选条件**：
- 出发地：选择出发机场
- 目的地：选择到达机场
- 日期范围：选择起止日期
- 权益卡类型：选择 666/2666/全部
- 航班号：输入航班号进行模糊搜索

### 编辑航班
1. 点击表格行的"编辑"按钮
2. 修改表单信息
3. 点击"保存"

**可编辑字段**：
- 航班号 ✓
- 出发地 ✓
- 目的地 ✓
- 起飞时间 ✓
- 到达时间 ✓
- 权益卡类型 ✓
- 可用座位 ✓
- 机型 ✓

### 删除航班
1. 点击表格行的"删除"按钮
2. 确认删除

或者批量删除：
1. 勾选多个复选框
2. 点击"批量删除"按钮
3. 确认删除

### 导出数据
1. 筛选需要的航班（可选）
2. 点击"导出 Excel"按钮
3. 文件自动下载

**导出包含的列**：
- 航班号
- 出发地
- 目的地
- 起飞时间
- 到达时间
- 权益卡类型
- 可用座位
- 机型
- 爬取时间

## 🔧 后端 API 速查

### 分页查询
```bash
GET /api/flights/paginated?page=1&pageSize=10&origin=北京首都
```

### 查询单条
```bash
GET /api/flights/1
```

### 更新航班
```bash
PUT /api/flights/1
{
  "availableSeats": 50
}
```

### 删除航班
```bash
DELETE /api/flights/1
```

### 批量删除
```bash
POST /api/flights/batch-delete
{
  "ids": [1, 2, 3]
}
```

## 🐛 常见问题

### Q: 表格无数据？
**A**:
1. 检查数据库中是否有航班数据
2. 尝试点击"重置"清空筛选条件
3. 检查浏览器控制台是否有错误

### Q: 编辑后没有保存？
**A**:
1. 检查表单是否有红色错误提示
2. 确保必填字段都已填写
3. 查看浏览器控制台是否有错误

### Q: 导出 Excel 不工作？
**A**:
1. 确保表格中有数据
2. 检查浏览器是否允许下载
3. 查看浏览器控制台是否有错误

### Q: 批量删除按钮灰色？
**A**:
需要先选中至少一条航班（勾选复选框）
