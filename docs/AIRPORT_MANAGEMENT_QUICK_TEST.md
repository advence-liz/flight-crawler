# 机场管理功能快速测试指南

## 测试前准备

### 启动服务
```bash
# 终端 1：启动后端
cd backend
npm run dev

# 终端 2：启动前端
cd frontend
npm run dev
```

### 访问页面
- 前端地址：http://localhost:5174/airport-management
- 后端 API：http://localhost:3000/api/airports

---

## 测试用例

### 测试 1：查看机场列表
**步骤**：
1. 打开机场管理页面
2. 观察统计卡片和表格数据

**预期结果**：
- ✅ 显示 4 个统计卡片（总数、启用、禁用、关联航班）
- ✅ 显示机场列表（默认 10 条/页）
- ✅ 表格包含列：机场名、城市、启用状态、发现时间、更新时间、操作

**验证**：
```bash
curl http://localhost:3000/api/airports/paginated
```

## 常见问题排查

### 问题 1：页面加载缓慢
**排查步骤**：
1. 检查后端是否正常运行：`curl http://localhost:3000/api/airports/paginated`
2. 检查浏览器网络标签页的加载时间
3. 检查数据库是否有大量数据

### 问题 2：编辑后数据未更新
**排查步骤**：
1. 刷新页面查看是否真的更新了
2. 检查浏览器控制台是否有错误
3. 检查后端日志

### 问题 3：导出 Excel 失败
**排查步骤**：
1. 检查是否有数据可导出
2. 检查浏览器是否允许下载
3. 检查磁盘空间是否充足
