# 航班管理功能 - 快速测试指南

## 快速开始

### 1. 启动应用

```bash
# 在项目根目录运行
./start.sh

# 或手动启动
# 终端 1: 启动后端
cd backend && npm run dev

# 终端 2: 启动前端
cd frontend && npm run dev
```

### 2. 访问页面

打开浏览器，访问 `http://localhost:5173/flight-management`

## API 调用验证

### 使用 curl 或 Postman 测试

#### 查询分页航班
```bash
curl -X GET 'http://localhost:3000/api/flights/paginated?page=1&pageSize=10&origin=北京首都'
```

**预期响应**：
```json
{
  "flights": [...],
  "total": 42,
  "page": 1,
  "pageSize": 10
}
```

#### 查询单条航班
```bash
curl -X GET 'http://localhost:3000/api/flights/1'
```

**预期响应**：
```json
{
  "id": 1,
  "flightNo": "HU1234",
  "origin": "北京首都",
  "destination": "上海浦东",
  ...
}
```

#### 更新航班
```bash
curl -X PUT 'http://localhost:3000/api/flights/1' \
  -H 'Content-Type: application/json' \
  -d '{"availableSeats": 50}'
```

**预期响应**：
```json
{
  "message": "更新成功",
  "flight": {...}
}
```

#### 删除航班
```bash
curl -X DELETE 'http://localhost:3000/api/flights/1'
```

**预期响应**：
```json
{
  "success": true,
  "message": "删除成功"
}
```

#### 批量删除航班
```bash
curl -X POST 'http://localhost:3000/api/flights/batch-delete' \
  -H 'Content-Type: application/json' \
  -d '{"ids": [1, 2, 3]}'
```

**预期响应**：
```json
{
  "success": true,
  "deletedCount": 3,
  "message": "成功删除 3 条航班"
}
```

## 常见问题排查

### 问题 1：页面无法加载
**症状**：打开 `/flight-management` 显示空白或 404

**排查步骤**：
1. 检查后端是否正常运行（访问 `http://localhost:3000/api/flights/paginated`）
2. 检查前端是否正常运行（检查浏览器控制台是否有错误）
3. 检查路由配置是否正确（`App.tsx` 和 `Layout.tsx`）

### 问题 2：表格无数据
**症状**：页面加载但表格为空

**排查步骤**：
1. 检查数据库中是否有航班数据
2. 检查浏览器网络面板，API 是否返回数据
3. 检查浏览器控制台是否有错误

### 问题 3：编辑失败
**症状**：点击编辑后无法保存

**排查步骤**：
1. 检查浏览器控制台是否有错误
2. 检查网络面板，PUT 请求是否成功
3. 检查后端日志是否有错误

### 问题 4：导出不工作
**症状**：点击导出按钮无反应

**排查步骤**：
1. 检查是否安装了 xlsx 库（`npm list xlsx`）
2. 检查浏览器控制台是否有错误
3. 检查表格是否有数据
