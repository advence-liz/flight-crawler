## 常用操作

### 1️⃣ 查询机场
```
位置：筛选表单
步骤：
  1. 输入筛选条件（可选）
  2. 点击"查询"按钮
  3. 表格更新显示结果
```

### 2️⃣ 编辑机场
```
位置：表格操作列
步骤：
  1. 点击"编辑"按钮
  2. 修改机场信息
  3. 点击"保存"按钮
  4. 显示"更新成功"提示
```

### 3️⃣ 快速切换启用状态
```
位置：表格"启用状态"列
步骤：
  1. 直接点击 Switch 组件
  2. 状态立即切换
  3. 显示"更新成功"提示
```

### 4️⃣ 查看航班统计
```
位置：表格操作列
步骤：
  1. 点击"统计"按钮
  2. 弹出统计窗口
  3. 查看航班数量统计
```

### 5️⃣ 删除机场
```
位置：表格操作列
步骤：
  1. 点击"删除"按钮
  2. 确认删除
  3. 显示"删除成功"提示
```

### 6️⃣ 批量删除
```
位置：表格行选择 + 操作按钮区
步骤：
  1. 勾选多个机场
  2. 点击"批量删除"按钮
  3. 确认删除
  4. 显示"成功删除 X 个机场"提示
```

### 7️⃣ 导出 Excel
```
位置：操作按钮区
步骤：
  1. 可选：先筛选需要导出的数据
  2. 点击"导出 Excel"按钮
  3. 自动下载文件
```

## 筛选条件速查

### 按城市筛选
```
输入框：城市
示例：北京
结果：显示北京的所有机场
```

### 按机场名筛选
```
输入框：机场名
示例：首都
结果：显示名称包含"首都"的机场
```

### 按启用状态筛选
```
下拉框：启用状态
选项：启用 / 禁用
结果：显示对应状态的机场
```

### 组合筛选
```
示例：城市=北京 + 启用状态=启用
结果：显示北京启用的机场
```

## 统计卡片说明

| 卡片 | 含义 | 更新时机 |
|-----|------|---------|
| 总机场数 | 数据库中所有机场 | 页面加载、删除后 |
| 启用数 | 启用爬虫的机场 | 筛选结果变化 |
| 禁用数 | 禁用爬虫的机场 | 筛选结果变化 |
| 关联航班 | 当前页机场的航班总数 | 分页变化 |

## 表格列说明

| 列名 | 说明 | 可操作 |
|-----|------|--------|
| 机场名 | 机场的标准名称 | 点击编辑 |
| 城市 | 机场所在城市 | 点击编辑 |
| 启用状态 | 是否启用爬虫 | 直接切换 |

| 更新时间 | 机场最后更新的时间 | 只读 |

## 数据导出格式

### Excel 列
1. 机场名
2. 城市
3. 启用状态（启用/禁用）
4. 发现时间（YYYY-MM-DD HH:mm:ss）
5. 更新时间（YYYY-MM-DD HH:mm:ss）

### 文件命名
```
机场数据_YYYY-MM-DD_HH-mm-ss.xlsx
示例：机场数据_2026-03-17_16-30-45.xlsx
```

## 性能提示

### ✅ 推荐做法
- 使用筛选条件缩小查询范围
- 分页查看大数据集
- 使用 Switch 快速切换状态（比编辑框快）

### ❌ 避免做法
- 一次加载所有机场数据
- 频繁进行大批量删除操作
- 导出大量数据到 Excel（可能较慢）

## 常见错误提示

| 提示 | 原因 | 解决方案 |
|-----|------|---------|
| "请输入机场名" | 编辑框机场名为空 | 输入有效的机场名 |
| "请选择要删除的机场" | 没有勾选任何机场 | 先勾选机场再删除 |
| "没有数据可导出" | 查询结果为空 | 修改筛选条件重试 |
| "更新失败" | 网络问题或服务器错误 | 检查网络，稍后重试 |

## 与其他功能的关系

### 航班管理 ↔ 机场管理
- 航班管理：查看和管理单个航班
- 机场管理：查看和管理机场及其关联航班
- 关系：一个机场可以有多个航班

### 数据管理 ↔ 机场管理
- 数据管理：爬取新的机场和航班数据
- 机场管理：管理已爬取的机场信息
- 关系：数据管理会自动发现新机场，机场管理可以控制哪些机场启用爬虫

## 故障排查速查表

| 问题 | 检查项 | 解决方案 |
|-----|--------|---------|
| 页面加载慢 | 后端是否运行 | 检查后端服务状态 |
| 数据不更新 | 是否刷新页面 | 按 F5 刷新 |
| 删除失败 | 网络连接 | 检查网络，重试 |
| 导出失败 | 数据是否存在 | 先筛选数据再导出 |
| 编辑无反应 | 浏览器控制台 | 查看错误信息 |
