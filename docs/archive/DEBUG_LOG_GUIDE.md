## 🐛 常见问题排查

### 问题 1: 权益卡选择失败
**日志特征**：
```
❌ 所有 5 种策略都失败，未找到权益卡: 666权益卡航班
```

**排查步骤**：
1. 查看 `页面元素分析` 日志，确认页面上有 `.search-item` 元素
2. 查看 `card-fail.png` 截图，确认页面是否正常加载
3. 检查网络连接是否正常

### 问题 2: 出发地选择失败
**日志特征**：
```
❌ 所有 4 种策略都失败，未找到出发地选择器
```

**排查步骤**：
1. 查看 `页面元素分析` 日志，确认有 `depBox` 元素
2. 查看 `origin-fail.png` 截图
3. 检查城市名称是否正确

### 问题 3: 查询按钮点击失败
**日志特征**：
```
❌ 所有 5 种策略都失败，未找到查询按钮
```

**排查步骤**：
1. 查看 `页面元素分析` 日志，确认有 `queryBtn` 元素
2. 查看 `query-fail.png` 截图
3. 检查前面的步骤是否都成功

## 🎯 性能优化建议

1. **策略顺序优化**：最常成功的策略放在前面
2. **超时时间调整**：根据实际网络情况调整 `waitForTimeout`
3. **并发控制**：避免同时爬取过多日期和权益卡组合
