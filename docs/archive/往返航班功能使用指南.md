# 往返航班功能使用指南

## 🎯 功能概述

往返航班查询功能已成功集成到目的地查询页面中，现在您可以：

1. ✅ **快速识别**：在目的地列表中直观地看到哪些目的地支持往返
2. ✅ **详细信息**：查看去程和返程的航班数量、可用日期
3. ✅ **完整展示**：点击详情查看所有往返航班的完整信息

---

## 🚀 快速开始

### 1. 启动项目

**方式一：一键启动（推荐）**
```bash
./start.sh
```

**方式二：分别启动**
```bash
# 终端 1：启动后端
cd backend
npm run dev

# 终端 2：启动前端
cd frontend
npm run dev
```

### 2. 访问应用

打开浏览器访问：**http://localhost:5173**

### 3. 使用功能

1. 在目的地查询页面，选择：
   - **出发地**：例如"北京首都"
   - **日期范围**：例如"2026-03-16 到 2026-04-16"
   - **权益卡类型**：例如"全部权益卡"

2. 点击**"查询"**按钮

3. 查看结果：
   - 目的地列表会显示"可往返"标签
   - 去程和返程航班数量一目了然
   - 点击"查看详情"查看完整往返航班信息

## 💡 使用场景

### 场景 1：周末短途游

**需求**：想周五晚上出发，周日晚上回来

**操作步骤**：
1. 选择出发地：北京首都
2. 选择日期范围：本周五到下周日
3. 查询结果
4. 查看带有"可往返"标签的目的地
5. 点击详情，选择周五晚上的去程航班和周日晚上的返程航班

### 场景 2：长假旅行规划

**需求**：国庆假期想去多个城市，需要确认每个城市是否支持往返

**操作步骤**：
1. 选择出发地：北京首都
2. 选择日期范围：国庆假期
3. 查询结果
4. 快速浏览"可往返"标签，筛选出支持往返的城市
5. 规划多城市行程路线

### 场景 3：灵活行程安排

**需求**：不确定返程日期，想看看返程航班的选择

**操作步骤**：
1. 选择出发地和目的地
2. 选择较宽的日期范围
3. 查看返程航班的可用日期
4. 对比不同日期的航班时间
5. 选择最方便的返程日期

## 📋 API 文档

### 1. 查询目的地（包含往返信息）

**接口**: `GET /api/flights/destinations`

**参数**:
- `origin` (必填): 出发地
- `startDate` (必填): 开始日期 (YYYY-MM-DD)
- `endDate` (必填): 结束日期 (YYYY-MM-DD)
- `flightType` (可选): 权益卡类型（全部/666权益卡航班/2666权益卡航班）

**响应示例**:
```json
{
  "destinations": [
    {
      "destination": "广州",
      "flightCount": 7,
      "availableDates": ["2026-03-17", "2026-03-18", ...],
      "cardTypes": ["666权益卡航班", "2666权益卡航班"],
      "hasReturn": true,
      "returnFlightCount": 14,
      "returnAvailableDates": ["2026-03-17", "2026-03-18", ...]
    }
  ],
  "totalCount": 10,
  "dateRange": {
    "start": "2026-03-16",
    "end": "2026-04-16"
  }
}
```

### 2. 查询往返航班详情

**接口**: `GET /api/flights/round-trip`

**参数**:
- `origin` (必填): 出发地
- `destination` (必填): 目的地
- `startDate` (必填): 开始日期 (YYYY-MM-DD)
- `endDate` (必填): 结束日期 (YYYY-MM-DD)
- `flightType` (可选): 权益卡类型

**响应示例**:
```json
{
  "outboundFlights": [
    {
      "id": 773,
      "flightNo": "HU7815",
      "origin": "北京首都",
      "destination": "广州",

      "arrivalTime": "2026-03-17T15:55:00.000Z",
```

## ❓ 常见问题

### Q1: 为什么有些目的地没有"可往返"标签？

**A**: 这表示在查询的日期范围内，该目的地没有返程航班。可能的原因：
- 该航线是单向航线
- 返程航班不在查询的日期范围内
- 返程航班不支持所选的权益卡类型

**解决方法**：尝试扩大日期范围或更换权益卡类型。

### Q2: 去程和返程的航班数量为什么不一样？

**A**: 这是正常的，因为：
- 不同方向的航班频次可能不同
- 某些日期可能只有去程或只有返程
- 不同权益卡类型的可用航班不同

### Q3: 点击"查看详情"后没有显示返程航班？

**A**: 这表示该目的地在查询的日期范围内确实没有返程航班。页面会显示"暂无返程航班"的提示。

### Q4: 如何查看特定日期的往返航班组合？

**A**:
1. 在详情页面中，去程和返程航班都按日期排序
2. 找到您想要的去程日期
3. 在返程航班中找到合适的返程日期
4. 记录航班号和时间，即可完成行程规划

### Q5: 往返航班的权益卡类型必须一致吗？

**A**: 不一定。查询时：
- 选择"全部权益卡"：可以看到所有权益卡类型的往返航班
- 选择特定权益卡：只显示该类型的往返航班

建议选择"全部权益卡"以获得更多选择。

如果遇到问题：

1. **查看日志**：
